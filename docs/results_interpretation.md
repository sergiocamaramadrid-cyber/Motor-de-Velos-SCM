# Results Interpretation

## Environmental figures

### ΔF3 vs environment

Figure: `results/fig_deltaF3_environment.png`

Interpret the linear-fit p-value as:

- `p < 0.01` → strong environmental signal
- `0.01 ≤ p < 0.05` → moderate evidence
- `p ≥ 0.05` → no significant evidence

### β vs environment

Figure: `results/fig_beta_environment.png`

This test evaluates whether the deep-regime slope β depends on outer HI
surface density (`logSigmaHI_out`).

Interpret the linear-fit p-value as:

- `p < 0.01` → strong environmental signal
- `0.01 ≤ p < 0.05` → moderate evidence
- `p ≥ 0.05` → no significant evidence
