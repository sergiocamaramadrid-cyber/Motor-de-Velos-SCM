# Jeans Equation Connection

The SCM observable

F3 = d log(v) / d log(r)

can be interpreted within the framework of the Jeans equation for
gravitational equilibrium.

For a stationary spherical system the Jeans equation reads

d(ρ σ_r^2)/dr + 2β_a ρ σ_r^2 / r = −ρ dΦ/dr

where Φ is the gravitational potential.

For rotation-dominated disks the circular velocity satisfies

v_c^2 = r dΦ/dr

so that the observable F3 can be written as

F3 = (1/2) [1 + d ln(dΦ/dr) / d ln r]

This shows that F3 directly measures the logarithmic gradient of the
gravitational field.

In the SCM framework the potential can be decomposed as

Φ = Φ_bar + Φ_env

so that the environmental contribution modifies the radial gradient of
the potential and therefore the observable F3.

Consequently the quantity ΔF3 provides an observational probe of
perturbations in the effective gravitational potential at large radii.
