# F3 Observable

`F3` y `f3_scm` representan el observable estructural del framework en el régimen profundo.

- `delta_f3 = f3_scm - 0.5` se usa como desviación respecto al valor esperado en el régimen profundo.
- `beta` y `friction_slope` cuantifican la pendiente efectiva asociada al ajuste físico local.

El catálogo debe incluir flags de calidad (`fit_ok`, `quality_flag`, `reliable`, `velo_inerte_flag`) para auditoría científica.
